# Perina Legal Pages

Static legal pages for GitHub Pages, matching the Focusly approach.

Intended public URLs after publishing this folder as a separate public repository named `perina-legal`:

- `https://nguyenvankhoecs50.github.io/perina-legal/privacy.html`
- `https://nguyenvankhoecs50.github.io/perina-legal/terms.html`

## Publish

1. Create a new public GitHub repository named `perina-legal`.
2. Copy `index.html`, `privacy.html`, and `terms.html` into the repository root.
3. Push to the `main` branch.
4. In GitHub, open Settings -> Pages.
5. Set source to `Deploy from a branch`, branch `main`, folder `/root`.
6. Wait for GitHub Pages to publish, then test both URLs above.

## Store Use

Use the Privacy Policy URL in:

- Google Play Console -> App content -> Privacy Policy
- App Store Connect -> App Privacy / App Information

Use the Terms URL anywhere the store asks for subscription terms, and in app paywall/profile/settings once legal links are added.
